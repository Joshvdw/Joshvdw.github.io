What is the difference between Margin, Border, and Padding?


In this blog I'll be explaining (in plain English) the difference is between these css attributes.

All of these attributes relate to the "box-model", which is essentially an invisible box that wraps around every HTML element.

Manipulating the box-model is how a developer structures & styles the layout of content.

Now to break down the difference between margin, border & padding, let's first start with margin - which is the outermost attribute of the box-model.

Margin measures the space on the outside of the box-model.

Border is the outline or stroke of the box-model.

Padding measure the space on the inside of the box-model.

Although all these attributes can be given a width or height in pixels or percentages, they all target a different area of the box-model (as pictured across).
